# PublicHealthTool
### Installation

1) Download node.js
2) Clone the repository  
`git clone https://github.com/vinhdoan1/PublicHealthTool.git`  
3) Install node modules
`npm install`  
4) Get Google Maps API Key and enter into congif.js:
`googlemapsapikey: "YOUR_KEY_HERE"`  
5) Run
`npm start`
